# Organización y Metodología

## Agile

### ¿Qué es Agile?

Agile es una metodología de desarrollo de software basada en la colaboración, la adaptación al cambio y las entregas incrementales de valor. Su objetivo es crear productos funcionales de forma continua y mejorar según las necesidades del cliente.

### Valores del Manifiesto Ágil

1. Individuos e interacciones sobre procesos y herramientas.
2. Software funcionando sobre documentación extensiva.
3. Colaboración con el cliente sobre negociación contractual.
4. Respuesta al cambio sobre seguir un plan establecido.

### Ventajas de Agile

* Permite adaptarse rápidamente a cambios en los requisitos.
* Favorece la comunicación entre los miembros del equipo.
* Reduce riesgos mediante entregas frecuentes.
* Mejora la satisfacción del cliente.

## Historia de Usuario

### Historia de Usuario

Como usuario de una tienda online, quiero añadir productos al carrito para poder comprarlos más tarde de forma rápida y sencilla.

### Tareas Técnicas

* Crear la estructura HTML del botón "Añadir al carrito".
* Diseñar el botón utilizando CSS.
* Programar la funcionalidad con JavaScript.
* Actualizar el contador del carrito.
* Mostrar un mensaje de confirmación al usuario.

---

## Criterios de Aceptación

La funcionalidad se considerará completada cuando:

* El botón "Añadir al carrito" sea visible en la página del producto.
* Al hacer clic en el botón, el producto se añada correctamente al carrito.
* El contador del carrito aumente en una unidad.
* El botón cambie de color al pasar el cursor por encima.
* Se muestre un mensaje de confirmación indicando que el producto fue añadido.
